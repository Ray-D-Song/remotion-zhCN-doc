Remotion的渲染引擎是使用Node.JS构建的，这使得在云中渲染视频变得容易。

## 在AWS Lambda上渲染视频

在云中渲染视频的最简单最快速的方法是使用[`@remotion/lambda`](/docs/lambda)。

## 使用Node.js API渲染视频

我们提供了一组API来使用Node.js和Bun来渲染视频。  
查看[示例](/docs/ssr-node)或[API参考](/docs/renderer)以获取更多信息。

## 使用GitHub Actions进行渲染

[Hello World入门模板](https://github.com/remotion-dev/template-helloworld)包括一个GitHub Actions工作流文件[`.github/workflows/render-video.yml`](https://github.com/remotion-dev/template-helloworld/blob/main/.github/workflows/render-video.yml)。

<Step>1</Step> 将模板提交到GitHub存储库。<br/>
<Step>2</Step> 在GitHub上，点击<code>Actions</code>选项卡。<br/>
<Step>3</Step> 在左侧选择<code>Render video</code>工作流。<br/>
<Step>4</Step> 应该会出现一个<code>Run workflow</code>按钮。点击它。<br/>
<Step>5</Step> 填写根组件的props并点击<code>Run workflow</code>。<br/>
<Step>6</Step> 渲染完成后，您可以在<code>Artifacts</code>下下载视频。<br/><br/>

请注意，运行工作流可能会产生费用。但是，只有在您主动触发时工作流才会运行。

另请参阅：[在GitHub Actions中传递输入props](/docs/passing-props#passing-input-props-in-github-actions)
