组合是可以渲染的内容。它包括：

<Step>1</Step> 一个 React 组件 <br />
<Step>2</Step> 画布的宽度和高度 <br/>
<Step>3</Step> “每秒帧数”（FPS）值 <br/>
<Step>4</Step> 一个持续时间值<br/>
<Step>5</Step> 一个标识符 <code>id</code><br/><br/>

可以通过渲染 [`<Composition>`](/docs/composition) 组件在 [Remotion Studio](/docs/terminology/studio) 中注册组合。  
持续时间为 1 帧的组合也被称为 [`<Still>`](/docs/still)。

在 [Remotion Player](/docs/terminology/player) 中，您不使用 `<Composition>` 组件，而是直接将组件和元数据传递给 [`<Player>`](/docs/player/player)。

不要与 [`<Sequence>`](/docs/terminology/sequence) 混淆。

## 组合标识符

您将作为 `id` 属性传递给 [`<Composition>` 组件](/docs/composition) 的字符串。  
您需要组合标识符来引用您想要渲染的内容，例如：`npx remotion render src/index <composition-id>`。
