入口点是[Remotion CLI](/docs/cli)和[`@remotion/renderer`](/docs/renderer) API将查找Remotion项目的文件。

- 在大多数模板中，默认情况下是`src/index.ts`。
- 在旧项目中，可能会使用`.tsx`扩展名而不是`.ts`。
- 入口点可以传递给渲染命令，例如：`npx remotion render src/index.ts`。
- 入口点应该调用[`registerRoot()`](/docs/register-root)。
- 如果使用[`npx remotion render`](/docs/cli/render)渲染视频，则入口点会以灰色打印出来。
- 您可以使用[`Config.setEntryPoint()`](/docs/config#setentrypoint)在配置文件中自定义默认入口点。

## 正在使用哪个入口点？

如果调用[`npx remotion render`](/docs/cli/render)，将打印出入口点以及选择它的原因。算法如下：

<Step>1</Step> 如果直接传递了入口点，例如<code>npx remotion render src/index.ts</code>，则使用该入口点。<br/>
<Step>2</Step> 否则，如果在<a href="/docs/config">配置文件</a>中使用<a href="/docs/config#setentrypoint"><code>Config.setEntryPoint()</code></a>指定了路径，则使用该路径。<br/>
<Step>3</Step> 遍历以下常见路径，如果存在其中一个，则使用该路径：

<ul>
<li><code>src/index.ts</code></li>
<li><code>src/index.tsx</code></li>
<li><code>src/index.js</code></li>
<li><code>src/index.mjs</code></li>
<li><code>remotion/index.tsx</code></li>
<li><code>remotion/index.ts</code></li>
<li><code>remotion/index.js</code></li>
<li><code>remotion/index.mjs</code></li>
</ul>

## 在播放器中

在[Remotion播放器](/docs/terminology/player)中，没有入口点的概念。  
您直接将React组件和元数据传递给[`<Player>`](/docs/player/player)。
