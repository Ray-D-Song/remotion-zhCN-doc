[`<Sequence>`](/docs/sequence) 是 Remotion 中的内置组件，具有以下功能：

- 绝对定位内容
- 允许您对元素进行时间偏移

将其视为 After Effects 或 Photoshop 中“图层”的等价物。  
不要与 [`<Composition>`](/docs/terminology/composition) 混淆。

## 布局

[`<Sequence>`](/docs/sequence) 默认情况下也会在 DOM 中绝对定位，因此您可以使用它将元素叠加在彼此之上。

## 时间偏移

考虑一个从第 0 帧开始动画的元素。  
如果您想要延迟动画，而不是重构动画，您可以将其包装在 `<Sequence>` 中，并使用 [`from`](/docs/sequence#from) 属性定义延迟。

使用序列，您还可以通过传递 [`durationInFrames`](/docs/sequence#durationinframes) 属性来裁剪动画的开始和结束。
