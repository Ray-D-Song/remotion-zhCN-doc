根文件是导出根组件的文件，该组件渲染一个或多个[`<Composition />`](/docs/composition)组件。  
默认情况下，在大多数 Remotion 模板中，它是 `src/Root.tsx` 或 `remotion/Root.tsx`，但您可以任意重新组织文件。

[入口点](/docs/terminology/entry-point)将导入根文件并使用[`registerRoot()`](/docs/register-root)进行注册。

查看[Hello World 项目的根文件](https://github.com/remotion-dev/template-helloworld/blob/main/src/Root.tsx)作为示例。
