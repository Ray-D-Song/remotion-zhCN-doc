[Remotion Studio](/docs/studio) 是在运行 [`npx remotion studio`](/docs/cli/studio) 时打开的编辑器。  
它允许快速编辑和播放 [合成](/docs/terminology/composition)。  
它包含在 [`@remotion/cli`](https://npmjs.org/packages/@remotion/cli) 包中。

不要与 [Remotion Player](/docs/player) 混淆。

您可以将 [Remotion Studio](/docs/studio/deploy-server) 部署到 VPS 上进行渲染。  
自 v4.0.93 版本以来，Studio 也嵌入在每个 [Remotion Bundle](/docs/terminology/bundle) 中，尽管它不支持渲染。

## Remotion 预览

Remotion Studio 的原名，在 v4.0.0 版本中更名。  
由于预览获得了比仅仅预览更多的功能，因此名称不再合适。
