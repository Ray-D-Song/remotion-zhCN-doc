Remotion Root 是 Remotion 命令执行的目录。  
它影响了 [public dir](/docs/terminology/public-dir)、`.env` 文件以及 [config file](/docs/config) 等的默认位置。

## Remotion Root 是哪个目录？

要确定 Remotion Root，请从执行命令或运行脚本的目录开始，向上移动，直到找到一个包含 `package.json` 文件的目录。  
如果传递 `--log=verbose`，许多 Remotion CLI 命令也会打印 Remotion Root 目录。
