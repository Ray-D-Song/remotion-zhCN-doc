Remotion Player 是 React 组件 [`<Player>`](/docs/player)，可以嵌入到 React 应用程序中。  
可以通过将 [`@remotion/player`](https://npmjs.org/packages/@remotion/player) 包安装到 React 应用程序中来使用它（例如：Create React App、Next.JS、Remix、Vite）。  
它与 [Remotion Studio](/docs/terminology/studio) 不同。
