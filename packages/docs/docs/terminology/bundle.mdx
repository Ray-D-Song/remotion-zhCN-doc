一旦您用 React 编写了您的视频，您需要使用 [Webpack](https://webpack.js.org/) 进行打包以便渲染它。  
打包过程的输出被称为 "bundle"。  
它是一个包含 HTML、CSS、JavaScript 和您的资源的文件夹。

当您使用命令行进行渲染时，会自动生成一个 bundle（这是 `Bundling video` 步骤）。

您可以使用 [`bundle()`](/docs/bundle) API 来以编程方式创建一个 bundle。

您可以使用 [`npx remotion bundle`](/docs/cli/bundle) 命令在 CLI 中创建一个 bundle。

要打包并部署到 S3，您可以使用 [`npx remotion lambda sites create`](/docs/lambda/cli/sites#create) 命令。

如果您将一个 bundle 托管在一个 URL 下，它就变成了一个 [Serve URL](/docs/terminology/serve-url)，您可以指定它来渲染视频。
