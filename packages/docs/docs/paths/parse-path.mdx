_属于[`@remotion/paths`](/docs/paths)包。从v3.3.40版本开始可用_

将SVG字符串路径解析为`Instruction`数组。

```tsx twoslash title="reset-path.ts"
import { parsePath } from "@remotion/paths";

const newPath = parsePath("M 10 10 L 20 20");

/*
  [
    { type: "M", x: 10, y: 10 },
    { type: "L", x: 20, y: 20 },
  ]
*/
```

如果SVG路径无效，此函数将抛出异常。

## 返回类型

一个`Instruction`数组。`Instruction`类型也可以从`@remotion/paths`中导入：

```ts twoslash
import type { Instruction } from "@remotion/paths";
```

该类型具有以下结构：

      y: number;
    }

      type: "S";

    }

      type: "A";

      dy: number;

      type: "v";

      type: "c";
      cp1dx: number;
      cp1dy: number;
      cp2dx: number;
      cp2dy: number;
      dx: number;
      dy: number;
    }
  | {
      type: "s";
      cpdx: number;
      cpdy: number;

      dy: number;
    }
  | {
      type: "q";
      cpdx: number;
      cpdy: number;
      dx: number;

  | {
      type: "t";

      dy: number;
    }

      type: "a";
      rx: number;

    }

    }

      type: "z";
    };
