_属于[`@remotion/paths`](/docs/paths)包。_

将路径反转，使结束和开始位置互换。

```tsx twoslash
import { reversePath } from "@remotion/paths";

const reversedPath = reversePath("M 0 0 L 100 0");
console.log(reversedPath); // "L 100 0 M 0 0"
```

如果路径无效，该函数将抛出错误：

```tsx twoslash
import { reversePath } from "@remotion/paths";
// ---cut---
reversePath("remotion"); // Error: Malformed path data: ...
```
