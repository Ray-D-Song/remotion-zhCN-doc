使用此钩子，您可以检索有关所在合成的一些信息。

## 示例

```tsx twoslash
import React from "react";
import { useVideoConfig } from "remotion";

export const MyComp: React.FC = () => {
  const { width, height, fps, durationInFrames } = useVideoConfig();
  console.log(width); // 1920
  console.log(height); // 1080
  console.log(fps); // 30;
  console.log(durationInFrames); // 300

  return <div>Hello World!</div>;
};
```

## 返回值

一个具有以下属性的对象：

### `width`

合成的宽度（以像素为单位），或者如果调用 `useVideoConfig()` 的组件是定义了宽度的 [`<Sequence>`](/docs/sequence) 的子组件，则为 [`<Sequence>`](/docs/sequence) 的 `width`。

### `height`

合成的高度（以像素为单位），或者如果调用 `useVideoConfig()` 的组件是定义了高度的 [`<Sequence>`](/docs/sequence) 的子组件，则为 [`<Sequence>`](/docs/sequence) 的 `height`。

### `fps`

合成的帧速率，以每秒帧数为单位。

### `durationInFrames`

合成的持续时间（以帧为单位），或者如果调用 `useVideoConfig()` 的组件是定义了 `durationInFrames` 的 [`<Sequence>`](/docs/sequence) 的子组件，则为 [`<Sequence>`](/docs/sequence) 的 `durationInFrames`。

### `id`

合成的 ID。这与 [`<Composition>`](/docs/composition) 组件的 `id` 属性相同。

### `defaultProps`

您在合成中定义为 `defaultProps` 的对象。

### `props`<AvailableFrom v="4.0.0" />

传递给合成的 props，在[所有转换](/docs/props-resolution)之后。

### `defaultCodec`<AvailableFrom v="4.0.54" />

用于渲染此合成的默认编解码器。使用 [`calculateMetadata()`](/docs/calculate-metadata) 进行修改。
