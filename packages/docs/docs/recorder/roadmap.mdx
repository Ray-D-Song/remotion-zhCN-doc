购买 Recorder 包括访问所有未来的更新。

以下是我们计划添加的内容列表。  
我们积极开发 Recorder，因为我们将其用于我们自己的视频，但我们不会为这些功能设定日期。

我们建议您只在当前状态下发现它有用或计划自行开发缺失功能时购买 Recorder。

## ✅ SRT 导出

YouTube 上的惯例是以 `.srt` 文件的形式提供字幕，而不是将其固定在视频中。  
我们计划将此功能作为 Remotion 的一个特性添加，然后在 Recorder 中启用它。

:::note
这已于 2024 年 6 月 23 日发布！
:::

## 9:16 格式

我们计划支持一种新格式，使视频在 Instagram、YouTube Shorts 和 TikTok 上感觉更加原生。

- 摄像头和显示器的不同布局
- 逐字字幕
- 字幕动画

## 改进编辑体验

我们计划添加新的 Remotion API 和 Remotion Studio 功能，使编辑比使用属性编辑器更容易。

## 给我们反馈

让我们在 [`#recorder`](https://remotion.dev/recorder) 频道上知道您希望下一步看到什么。
