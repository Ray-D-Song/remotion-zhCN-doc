您还可以从其他来源添加录音，比如您的手机。  
只需将文件命名为 `webcam`，后面跟上任何数字，比如 `10`，再跟上支持的文件扩展名之一：`.mp4`、`.webm`、`.mkv` 或 `.mov`。

您的录音在视频中出现的顺序是基于数字后缀的。  
以下是您可以组织录音的一种方式：

```txt
public
└── my-video
    ├── webcam10.mp4
    ├── display10.mp4
    ├── webcam20.mp4
    └── webcam30.mp4
```

如果只有一个带有该数字的 `webcam` 录音，则摄像头将全屏显示。

只有在还有一个 `webcam` 录音时，才会捕捉到 `display` 录音。  
在这种情况下，摄像头将被缩小显示。

如果您只录制了显示内容，请仍然为其添加 `webcam` 前缀，以便捕捉到它。
