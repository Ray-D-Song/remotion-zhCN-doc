有时您只想录制屏幕的一部分。  
例如：您不希望录制您的 Mac 的 dock 和菜单栏。

实现这一目标的最佳方法是使用 [OBS](https://obsproject.com/) 创建虚拟摄像头。

<Step>1</Step> 从他们的网站下载 <a href="https://obsproject.com/">
  OBS
</a> 并安装它。
<br />

<Step>2</Step> 打开 OBS 并单击 <kbd>设置</kbd> -> <kbd>视频</kbd>。
将“基本分辨率”和画布分辨率都设置为您期望的输出分辨率。我们建议使用 1920x1080。单击 <kbd>确定</kbd>。
<br />
<Step>3</Step> 在 <kbd>来源</kbd> 下，添加您的显示器。
<br />
<Step>4</Step> 通过拖动视频的角落来裁剪视频。
<br />
<Step>5</Step> 单击 <kbd>启动虚拟摄像头</kbd>。<br />
<Step>6</Step> 在 Remotion 的 <a href="/docs/recorder/record/">
  录制界面
</a> 中，您现在可以选择“OBS 虚拟摄像头”作为来源。
