要删除录音，请在文件资源管理器中打开项目，并在 `public` 文件夹中找到录音：

```
public
└── my-video
    ├── webcam10.mp4
    ├── display10.mp4
    ├── webcam20.mp4
    └── webcam30.mp4
```

找到要删除的录音并将其删除。  
Remotion Studio 将自动刷新并将所有其他录音移动到下一个可用编号。

为了防止场景设置级联，还要在默认属性编辑器中删除场景。
