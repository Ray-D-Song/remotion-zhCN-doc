要生成字幕，Remotion Recorder 使用 [Whisper.cpp](https://github.com/ggerganov/whisper.cpp) 进行快速准确的转录。
每次使用 Remotion Recorder 录制剪辑时，字幕会自动生成并保存到与您的录音相同的文件夹中。

## 安装 Whisper.cpp

第一次完成录制剪辑时，Whisper.cpp 和一个 1.5GB 的模型将会安装到您的计算机上。这可能需要几分钟的时间。
安装完成后，将会生成 `webcam` 剪辑的字幕。

:::note
仅为带有 `webcam` 前缀的文件生成字幕。
:::

## 进行更正

如果 AI 出现错误，没问题，有多种方式可以手动更正转录。[查看这里如何操作](/docs/recorder/editing/captions)。

## 通过 CLI 生成字幕

对于[外部录制](/docs/recorder/external-recordings)，您也可以通过 CLI 生成字幕。

```
bun sub.ts
```

请注意，您想转录的文件的名称需要以 `webcam` 前缀开头，其他文件将被忽略。
包含字幕的 JSON 文件将会生成并保存在 `public/<composition-id>/sub[timestamp].json` 下。

## 非英语语言

如果您不是用英语录制，请编辑 `config/whisper.ts` 文件。

将语言设置为 [支持的值](/docs/install-whisper-cpp/transcribe#language)，并将 `model` 更改为不以 `.en` 结尾的 [支持的值](/docs/install-whisper-cpp/download-whisper-model#model)。
建议在非英语语言中转录时选择一个更大的模型。
