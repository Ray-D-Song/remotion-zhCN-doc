Remotion Recorder 是一款首创的基于 JavaScript 的视频创建工具，您可以控制源代码。

## 所需技能

要使用 Recorder 创建视频，您需要具备以下技能：

- 克隆 Git 存储库
- 在命令行中执行命令
- 在代码编辑器中打开项目

作为开发人员的技能越高，您就越能够根据自己的需求定制 Recorder。

## 我需要懂编程吗？

要以默认样式创建视频，您无需编程。  
但是，由于可以通过修改代码进行自定义，因此 Recorder 对开发人员很有吸引力。

## 面向对象群体

Recorder 面向希望完全掌控其工作流程并认为使用代码生成视频的灵活性具有价值的内容创作者。

它最适合那些拍摄自己的面部和屏幕以创建技术内容（屏幕录制和课程）的创作者。

Recorder 的功能集与传统视频编辑器不相似。
