# 设置

首先，[购买](/docs/recorder/buy) 录制器以[获取存储库访问权限](https://github.com/remotion-dev/recorder)。

## 1. 复制和克隆

[创建一个分支](https://github.com/remotion-dev/recorder/fork) 从 `remotion-dev/recorder` 存储库。  
使用 `git clone` 克隆存储库并进入该目录。

:::note
您需要将您的分支保持**私有**，以防泄漏 Remotion Recorder 的源代码。
:::

然后，克隆您的分支以在本地获取项目。

## 2. 安装 Bun

Remotion Recorder 需要 [Bun](https://bun.sh/)，最低版本为 1.1.11。要安装它，请运行

```bash
curl -fsSL https://bun.sh/install | bash
```

## 3. 安装依赖项

一旦安装了 Bun，运行

```bash
bun i
```

以安装所有必要的依赖项。

## 4. 安装 Whisper.cpp

_可选_

现在是初始化 Whisper.cpp 以进行字幕处理的好时机。  
这将安装 Whisper.cpp 并下载模型（默认大小为 1.5GB）。

```
bun sub.ts
```

## 5. 运行 Remotion Recorder

完成后，您可以开始运行 Remotion Recorder：

```bash
bun start
```
