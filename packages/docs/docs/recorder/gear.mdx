要使用录制设备，您只需要一台带有摄像头和麦克风的笔记本电脑。  
如果您想提升制作水平，有无限的可能性。

这是 Jonny 用来录制视频的设置。

## 电脑

使用 Windows 电脑来录制视频素材。  
这样可以减轻录制视频的压力，同时保持笔记本电脑的速度，并展示其屏幕。

## 相机

Panasonic Lumix S5 搭载 20-60 毫米镜头，安装在三脚架上。  
相机通过 HDMI 线连接到电脑，HDMI 线插入 Elgato Cam Link 4K 采集卡。

使用 [Lumix Tether](https://av.jpn.support.panasonic.com/support/global/cs/soft/download/d_lumixtether.html) 应用程序从电脑设置对焦和曝光。

## 屏幕

M2 Macbook Air 的屏幕通过 HDMI 线连接到 Elgato Game Capture HD60X 进行录制。

由于这也捕捉到菜单栏和 macOS dock，视频源被添加到 OBS 中，然后被裁剪为 16:9。

然后 Remotion Recording 接口使用 OBS 虚拟摄像头作为源。

## 麦克风

RØDE NT-USB 麦克风插入电脑。

麦克风安装在 RØDE PSA1 机械臂上。

## 灯光

两个 Amaran 200x S 灯光被用来从两侧照亮 Jonny。  
它们各自包裹在 Aputure Light Dome Mini II（54.61厘米）软箱中。
