# 布局

目前有 `landscape` 和 `square` 两种布局可供选择，用于定义视频的宽高比。

<br />
<br />
<div
  style={{
    display: "flex",
    justifyContent: "space-around",
    gap: 10,
    alignItems: "center",
    maxWidth: "72rem",
  }}
>
  {/* <Layout type={"portrait"} aspectWidth={9} aspectHeight={16} /> */}
  <Layout type={"landscape"} aspectWidth={16} aspectHeight={9} />
  <Layout type={"square"} aspectWidth={1} aspectHeight={1} />
</div>
<br />
<br />

要更改布局，请在右侧边栏中更改 `canvasLayout` 属性。
