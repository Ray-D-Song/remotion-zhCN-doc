## 修正

如果 AI 在字幕中出现错误，有三种不同的方法可以进行更正：

<Step>1</Step>在 Remotion Studio 中，您可以通过单击视频中的错误字幕来进行更正。这将打开一个文本编辑器，其中包含字幕，您可以直接编辑它们。完成更正后，单击编辑器右下角的“完成”。这将保存您所做的更改到包含您的字幕的 JSON 文件中。
<br />
<br />

<Step>2</Step>您可以直接使用您选择的编辑器编辑位于 <code>public/[your-composition-id]</code> 中的 JSON 文件。
<br />
<br />
<Step>3</Step>如果 AI 多次出现相同错误，您还可以通过在 <code>config/autocorrect.ts</code> 文件中添加逻辑到 <code>autocorrectWord</code> 函数来以编程方式重新映射单词。例如：

```tsx
const autocorrectWord = (word: Word): Word => {
  // Replace a single word with another one
  if (word.word === " github") {
    return {
      ...word,
      word: word.word.replace("github", " GitHub"),
    };
  }

  return word;
};
```

## 突出显示单词

要突出显示特定单词，请单击字幕中所需的单词。在打开的编辑器中，选择 `monospace` 选项以应用等宽字体和默认蓝色到所选单词。
如果您希望更改突出显示的颜色，请修改 `config/themes.ts` 文件中的 `WORD_HIGHLIGHT_COLOR` 常量。
