要添加新场景，请单击右侧边栏中的 `+`。

### 标题场景

标题由主标题和可选副标题组成。
可以通过在右侧勾选 `null` 复选框来删除副标题。

### 视频场景

视频场景包含摄像头录制、可选屏幕录制和字幕，如果有的话[生成](/docs/recorder/captions)。使用录制器捕获的每个录制对应一个视频场景。
对于每个新添加的视频场景，按时间顺序下一个录制将添加到时间轴上。

摄像头可以放置在视频的四个角之一。其他组件将自动围绕摄像头定位。

### 结束画面

在结束画面中，您可以添加行动呼叫，例如链接到您的社交媒体资料或其他网站。
