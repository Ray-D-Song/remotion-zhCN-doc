当生成字幕时，将使用 Whisper.cpp 时间戳来确定转录的开始和结束。

因此，在您手动修剪视频之前，应确保生成字幕。

## 编辑转录以纠正错误

修正错误的最简单方法是[单击字幕](/docs/recorder/editing/captions)并编辑转录。  
第一个非空令牌确定开始修剪。  
最后一个非空令牌确定结束修剪。

## 应用偏移量

在右侧的默认属性编辑器中，对场景应用 `startOffset`。  
偏移量以帧为单位。

负偏移将使视频提前开始，正偏移将使视频稍后开始。

要更改结束修剪，请对场景应用 `endOffset`。其逻辑与开始偏移相同。

## 自定义逻辑

确定开始和结束时间戳的逻辑在 [`remotion/calculate-metadata/add-metadata-to-scene.ts`](https://github.com/remotion-dev/recorder/blob/main/remotion/calculate-metadata/add-metadata-to-scene.ts#L56) 中。

默认的开始填充为 0.25 秒，默认的结束填充为 0.5 秒。

这些值在 [`remotion/calculate-metadata/get-start-end-frame.ts`](https://github.com/remotion-dev/recorder/blob/main/remotion/calculate-metadata/get-start-end-frame.ts#L8) 中定义。
