要开始编辑环境，请运行

```
bun start
```

这将启动开发服务器并在新窗口中打开一个 Remotion Studio 实例。

## 创建新合成

合成是您可以编辑和渲染的视频。

在左侧边栏中，通过右键单击 `empty` 合成并选择 `Duplicate` 来创建一个新合成，并为其指定一个 ID。  
ID 应与您存储视频的 `public` 子文件夹的名称相同。
**示例**：如果您将录制存储在 `public/hello-world` 中，则 ID 应为 `hello-world`。

## 向合成添加新场景

打开右侧边栏。默认情况下没有场景。  
单击 `+` 按钮添加新场景。

<img style={{ width: 300 }} src="/recorder/add-scene.png" />

默认情况下，它将添加一个视频场景，该场景将使用 `public/[composition-id]` 文件夹中的第一个录制。

您还可以将场景类型切换为其他类型，如 `endcard` 或 `title`。  
每种场景类型都有自己的设置，您可以在右侧边栏中进行配置。

要保存场景，请单击磁盘图标或使用 `Ctrl+S` / `Cmd+S`。
