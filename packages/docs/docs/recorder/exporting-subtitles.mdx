## 非横向布局

如果您以 `square` 布局导出视频，则字幕将被烧录到视频中。

这样做的理念是，方形视频通常发布在像 x.com 和 LinkedIn 这样的平台上，在那里视频会出现在一个静音的动态流中。  
重要的是，字幕始终可见，并且具有较大的字体大小。

## 横向布局

如果您以 `landscape` 布局导出视频，则字幕将默认导出为 `.srt` 文件，位于 `out/[composition-id]/captions.srt`。  
这是由于在 [`remotion/captions/srt/EmitSrtFile.tsx`](https://github.com/remotion-dev/recorder/blob/main/remotion/captions/srt/EmitSrtFile.tsx) 中使用了 [`<Artifact>`](/docs/artifact) API。

在上传到 YouTube 时，您应该同时上传 SRT 文件和视频，以便观众可以使用 YouTube 的原生功能切换字幕并调整字体大小。

## 烧录字幕

如果您想在横向布局的视频中烧录字幕，请编辑 [`remotion/captions/srt/SrtPreviewAndEditor/SrtPreviewAndEditor.tsx`](https://github.com/remotion-dev/recorder/blob/main/remotion/captions/srt/SrtPreviewAndEditor/SrtPreviewAndEditor.tsx) 文件，并删除以下行：

```diff title="remotion/captions/srt/SrtPreviewAndEditor/SrtPreviewAndEditor.tsx"
-  // During rendering, you will get the actual .srt file instead of the preview.
-  if (getRemotionEnvironment().isRendering) {
-    return null;
-  }
```

现在您的视频将包含字幕作为视频顶部的叠加层。
