如果您无法获得赏金，我们会通知您。  
我们不会将问题分配给您，而是会将您发送到此页面。

## 无法获得赏金的常见原因

我们发布赏金以解决问题，以便我们能够比自己解决所有问题更快地前进。  
为了提高效率，我们排除了以下情况的贡献者有资格获得赏金：

- 选择与自己技能不符的问题
- 在较长一段时间内没有提供状态更新
- 过分依赖人工智能解决方案
- 提交未经测试或有缺陷的解决方案
- 在我们看来，不能让我们更快前进

## 为什么我无法获得赏金？

这并不意味着您做得不好。  
我们旨在为赏金支付公平金额，但也更具选择性。

这意味着即使您的解决方案是合法的，您也可能变得无法获得赏金。

由于人工智能生成的提交不幸地增加，我们需要广泛排除人员 - 即使有时我们会犯错。

由于赏金的目标是加速 Remotion 的开发，因此没有救济措施，我们也无法提供具体反馈。

## 如何继续？

您并非被禁止为 Remotion 做出贡献。  
如果您仍然想解决问题，可以再次发表评论！

否则，您无需采取任何行动 - 另一个人可以接手这个问题。
