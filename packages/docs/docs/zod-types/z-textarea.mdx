使用此类型，您可以在[模式](/docs/schemas)中指定一个字符串，该字符串将显示为 Remotion Studio 中的多行文本区域。

使用 `zTextarea()` 时，即使用户输入多行，值也将是一个字符串。

要在组件中显示值而不丢失换行符，您可以在显示值的元素上使用 `white-space: pre-line` CSS 属性。
