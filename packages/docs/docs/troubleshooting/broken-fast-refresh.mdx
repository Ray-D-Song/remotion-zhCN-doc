如果 Remotion Studio 在您更新文件后没有更新，通常是由于以下两个原因之一：

<Step>1</Step> Studio 服务器断开连接：当您退出运行 <code>npx remotion studio</code> 时启动的进程时，通常是使用 <code>Ctrl+C</code> 或退出终端时会发生这种情况。<br/><br/>

<Step>2</Step> 文件名大小写不匹配：确保在导入时文件名的大小写正确。例如，如果您的文件名是 <code>MyComp.tsx</code>，但您使用 <code>{'import {MyComp} from "./myComp.tsx"'}</code> 进行导入，可能会导致 Fast Refresh 出现问题。<br/><br/>

:::note
这是 Webpack 中的一个 bug，我们已经[提交了一个问题](https://github.com/webpack/watchpack/issues/228)。
:::
