如果您收到错误消息：

```
A delayRender() "Loading root component" was called but not cleared after 28000ms
```

您已指定一个[入口点](/docs/terminology/entry-point)，但未调用[`registerRoot()`](/docs/register-root)。

在大多数模板中，入口点是 `src/index.ts`。

- 可能您已指定要呈现的组合列表（在大多数模板中是 `src/Root.tsx`）。
- 可能您已指定要呈现的组件的文件名。

确保将调用[`registerRoot()`](/docs/register-root)的文件作为入口点传递。

在 CLI 中，入口点作为参数传递给 render 命令：

```
npx remotion render [entry-point]
```

在 [`bundle()`](/docs/bundle) 和 [`deploySite()`](/docs/lambda/deploysite) Node.JS API 中，您通过 [`entryPoint`](/docs/terminology/entry-point) 属性传递入口点。
