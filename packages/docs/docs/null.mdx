<ExperimentalBadge>
该组件目前是技术验证。

API 可能会在任何版本中更改。升级时，请监视此文档页面以查看重大更改。
</ExperimentalBadge>

这是一个特殊组件，当渲染时，将完全跳过渲染帧。  
您可以在确定帧为空并节省渲染时间时进行渲染。

```tsx twoslash title="EmptyFrame.tsx"
import { AbsoluteFill, Experimental } from "remotion";

export const MyComp: React.FC = () => {
  return (
    <AbsoluteFill>
      <Experimental.Null />
      <div>This will not be rendered</div>
    </AbsoluteFill>
  );
};
```

## 规则

<div>
<Step>1</Step> 如果渲染此组件，则在渲染过程中不会拍摄屏幕截图
</div>
<div>
<Step>2</Step> 在 Remotion Studio 和 Remotion Player 中，内容将以零不透明度渲染。
</div>
<div>
<Step>3</Step> 即使在 <code>{"<Null>"}</code> 组件之外的元素也会消失。
</div>
<div>
<Step>4</Step> 音频仍将被渲染。
</div>
<div>
<Step>5</Step> 效果仍将被执行。
</div>

<div>
<Step>6</Step> 如果 <code>imageFormat</code> 是 <code>jpeg</code>，将生成黑色帧；如果 <code>imageFormat</code> 是 <code>png</code>，将生成透明帧。
</div>
<div>
<Step>7</Step> 每帧只能渲染一个 <code>{"<Experimental.Null>"}</code> 或 <a href="/docs/clipper"><code>{"<Experimental.Clipper>"}</code></a> 组件。  
 渲染多个将导致错误。
</div>
