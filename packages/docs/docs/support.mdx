Remotion团队**不提供**支持保证，您不应期望获得任何帮助。

但是，我们欢迎您提出问题，并投入大量时间来回答。  
如果您有问题需要解答，请按照以下步骤操作。

## 如何寻求帮助

[在此处查看](/docs/get-help)可以寻求帮助的渠道以及提问的最佳实践。

## 优先支持

公司许可证客户可以在Discord上为每个订阅的开发者席位获得一个 <span style={{
  fontWeight: "bold",
  color: "#0B84F3"
}}><img style={{height: 20, verticalAlign: "middle"}} src="/img/blue-heart.png"/> 公司许可证</span> 徽章。该徽章可在[remotion.pro](https://www.remotion.pro)上激活。

如果在公共频道发布，Remotion团队将尝试优先处理您的问题，尽管不提供支持保证。

我们仍然期望公司许可证订阅者遵循上述提问最佳实践。

## 付费咨询

我们的[Remotion专家](/experts)可提供技术建议和指导。凭借他们在创建Remotion项目方面的丰富经验，他们一定会提供极大帮助。

您还可以在此处请求与Remotion的创始人进行咨询会话[这里](https://cal.com/remotion/consulting)。  
这是唯一一种通过视频通话获得Remotion团队支持的方式。

## 当我们无法提供帮助时

如果：

- 未遵循上述准则
- 未付出足够的努力首先自行解决问题
- 您的问题缺乏重现或信息
- 问题的根源是由于React或JavaScript技能不足
- 违反了我们的[行为准则](https://remotion.dev/coc)

Remotion面向开发人员，因此预期具备React和Web开发的基本技能。  
您还应该注意这些技术的局限性，因为我们无法帮助您克服这些问题。
