您可以通过可视化方式编辑组件的默认属性，首先要使用您的[`<Composition>`](/docs/composition)注册一个[`schema`](/docs/composition#schema)。

## 打开默认属性编辑器

要打开默认属性编辑器，请按下右上角的<svg style={{width: 20, verticalAlign:"middle", marginLeft: 4, marginRight: 4}} viewBox="0 0 100 100" fill="transparent"><rect x="3" y="3" width="94" height="94" rx="7" stroke="var(--ifm-font-color-base)" strokeWidth="6"/><path d="M59 91.5V8.5C59 4.63401 62.134 1.5 66 1.5H92C95.866 1.5 99 4.63401 99 8.5V91.5C99 95.366 95.866 98.5 92 98.5H66C62.134 98.5 59 95.366 59 91.5Z" fill="var(--ifm-font-color-base)" stroke="var(--ifm-font-color-base)"/></svg>
图标。或者，按<kbd>Cmd/Ctrl</kbd> + <kbd>J</kbd> 切换侧边栏。选择<code>Props</code>选项卡。

## 编辑默认属性

使用属性编辑器中的控件实时更新您的合成的默认属性。如果有更改，将出现一个撤销 ↩️ 按钮，您可以使用它来恢复到代码中的更改。

## 支持的控件

已实现以下控件：

- `z.object()`
- `z.string()`
- `z.date()`
- `z.number()`
- `z.boolean()`
- `z.array()`
- `z.union()`（仅两种类型的联合，其中之一是`z.null()`或`z.undefined()`）
- `z.optional()`
- `z.nullable()`
- `z.enum()`
- [`zColor()`](/docs/zod-types/z-color)（来自`@remotion/zod-types`）
- [`zTextarea()`](/docs/zod-types/z-textarea)（来自`@remotion/zod-types`）
- [`staticFile()`](/docs/staticfile) 通过将其类型定义为`z.string()`并在代码中使用`staticFile()`来处理静态文件
- `.default()`

## 编辑 JSON

或者，您可以通过在属性编辑器中按下`JSON`来直接编辑JSON模式。如果模式无效，则更改不会生效。 



## 将默认 props 保存到您的代码中

Remotion 可以静态分析您的根文件，并允许您通过 💾 按钮将 props 保存回代码中。为了使其工作，您的默认 props 必须内联到您的 [`<Composition>`](/docs/composition) 中：

```tsx twoslash title="Inlined defaultProps" {15-18}
// @filename: MyComponent.tsx
import React from "react";
import { z } from "zod";

export const myCompSchema = z.object({
  propOne: z.string(),
  propTwo: z.string(),
});

export const MyComponent: React.FC<z.infer<typeof myCompSchema>> = ({
  propOne,
  propTwo,
}) => {
  return (
    <div>
      <h1>{propOne}</h1>
      <h2>{propTwo}</h2>
    </div>
  );
};

// @filename: Root.tsx
// organize-imports-ignore
// ---cut---
import React from "react";
import { Composition } from "remotion";
import { MyComponent, myCompSchema } from "./MyComponent";

export const RemotionRoot: React.FC = () => {
  return (
    <Composition
      id="my-video"
      component={MyComponent}
      durationInFrames={100}
      fps={30}
      width={1920}
      height={1080}
      schema={myCompSchema}
      defaultProps={{
        propOne: "Hello World",
        propTwo: "Welcome to Remotion",
      }}
    />
  );
};
```

## 从您的输入中渲染视频

如果您想要根据您的输入渲染视频，则无需将 props 保存回代码中。使用 `Render` 按钮打开渲染对话框，在那里您修改后的 [默认 props 将作为输入 props 填充](/docs/props-resolution)。

## 在视觉上更改输入 props

在 Props 编辑器中，您只能通过视觉方式编辑 [默认 props](/docs/props-resolution)。然后，您有机会使用 [`calculateMetadata()`](/docs/calculate-metadata) 函数覆盖它们。  
在 [渲染对话框](/docs/studio) 中，您有机会更改输入 props。\_\_  
它们也可以使用 [`calculateMetadata()`](/docs/calculate-metadata) 进行覆盖。
