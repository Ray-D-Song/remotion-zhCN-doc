如果您是以下之一，Remotion 是**免费**的：

- 个人
- 具有最多 3 名员工的营利组织
- 非营利组织或非盈利组织
- 正在评估 Remotion 是否适合您，并且尚未以商业方式使用它

如果您不符合以上任何条件，您需要获取公司许可证。访问 [remotion.pro](https://remotion.pro) 了解定价和购买许可证。

如果有疑问，请在使用 Remotion 之前阅读 [LICENSE 文件](https://github.com/remotion-dev/remotion/blob/main/LICENSE.md) 和 [公司许可证常见问题](https://www.remotion.pro/faq)。如果您需要澄清是否可以免费在项目中使用 Remotion，请联系 [我们](mailto:hi@remotion.dev)。
