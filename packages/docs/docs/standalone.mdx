Remotion API 被设计为尽可能纯粹，一次只做一件事。  
这使得它们不仅易于在 Remotion 中使用，也可以在其他环境中使用。

**加粗的 API** 在 [Remotion 许可证](https://remotion.dev/license) 下授权使用（公司需要购买许可证才能使用）。  
未加粗的 API 在 MIT 许可证下授权使用。
