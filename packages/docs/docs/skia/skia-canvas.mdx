一个[React Native Skia `<Canvas />` 组件](https://shopify.github.io/react-native-skia/docs/canvas/overview)，用于包装 Remotion 上下文。

您可以在其中放置来自 `@shopify/react-native-skia` 的元素！

```tsx twoslash
import { SkiaCanvas } from "@remotion/skia";
import { Fill } from "@shopify/react-native-skia";
import React from "react";
import { useVideoConfig } from "remotion";

const MySkiaVideo: React.FC = () => {
  const { width, height } = useVideoConfig();
  return (
    <SkiaCanvas width={width} height={height}>
      <Fill color="black" />
    </SkiaCanvas>
  );
};
```

## 属性

### `width`

画布的宽度（以像素为单位）。

### `height`

画布的高度（以像素为单位）。

### 继承的属性

所有被 [`<Canvas>`](https://shopify.github.io/react-native-skia/docs/canvas/overview) 接受的属性也同样被接受。
