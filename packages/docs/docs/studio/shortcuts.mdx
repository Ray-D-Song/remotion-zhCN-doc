## 键盘导航

菜单支持键盘导航。按下 <kbd>Tab</kbd> 在菜单之间导航，按下 <kbd>Enter</kbd> 选择一个项目。

## 快速切换器

按下 <kbd>⌘/Ctrl</kbd> + <kbd>K</kbd> 打开快速切换器。您可以使用它快速导航到项目中的任何文件。

## 禁用键盘快捷键

使用 [`--disable-keyboard-shortcuts`](/docs/cli/studio#--disable-keyboard-shortcuts) 启动 Studio 以禁用所有键盘快捷键。如果您将其嵌入到 iframe 中，这将非常有用。
