# deleteStaticFile()<AvailableFrom v="4.0.154"/>

从[`public`目录](/docs/terminology/public-dir)中删除文件。  
此API对于在[Remotion Studio](/docs/terminology/studio)中构建交互式体验非常有用。

## 示例

```tsx twoslash title="删除 'video.webm'"
import React, { useCallback } from "react";
import { deleteStaticFile } from "@remotion/studio";

export const DeleteStaticFileComp: React.FC = () => {
  const deleteFile = useCallback(async () => {
    const { existed } = await deleteStaticFile("video.webm");

    console.log(`Deleted file (${existed ? "existed" : "did not exist"})`);
  }, []);

  return <button onClick={deleteFile}>Delete</button>;
};
```

## 规则

<Step>1</Step> 只能在Remotion Studio中使用此API。
<br />
<Step>2</Step> 文件路径必须相对于<a href="/docs/terminology/public-dir">
  <code>public</code>目录
</a>。
<br />
<Step>3</Step> 不允许删除<a href="/docs/terminology/public-dir">
  <code>public</code>目录之外的文件
</a>。<br />
<Step>4</Step> 要删除子文件夹中的文件，请使用正斜杠<code>
  /
</code>，即使在Windows上也是如此。
<Step>5</Step> 您可以，但不必将文件路径包装在<a href="/docs/staticfile">
  <code>staticFile()</code>
</a>函数中。
