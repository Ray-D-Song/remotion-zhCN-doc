按下 <kbd>⌘/Ctrl</kbd> + <kbd>K</kbd> 来调用快速切换器。默认情况下，它会让您选择一个[合成](/docs/terminology/composition)。  
在输入字段中，通过键入它们的[ID](/docs/terminology/composition#composition-id)来过滤合成。

## 浏览菜单项

如果您在第一个字符处键入"`>`"，您可以浏览菜单项。  
在"`>`"后添加更多字符以过滤菜单项。

## 搜索文档

如果您在第一个字符处键入"`?`"，您可以搜索这份文档。  
在"`?`"后添加更多字符以输入查询。

## 键盘导航

使用上下箭头键浏览菜单项。  
按下 <kbd>Enter</kbd> 来选择当前高亮的项目并关闭快速切换器。  
按下 <kbd>Esc</kbd> 来关闭快速切换器。
