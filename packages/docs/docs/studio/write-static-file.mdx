# writeStaticFile()<AvailableFrom v="4.0.147"/>

将一些内容保存到[`public`目录](/docs/)中的文件中。  
此 API 对于在[Remotion Studio](/docs/terminology/studio)中构建交互式体验非常有用。

## 示例

```tsx twoslash title="将 'Hello world' 写入 public/file.txt"
import React, { useCallback } from "react";
import { writeStaticFile } from "@remotion/studio";

export const WriteStaticFileComp: React.FC = () => {
  const saveFile = useCallback(async () => {
    await writeStaticFile({
      filePath: "file.txt",
      contents: "Hello world",
    });

    console.log("Saved!");
  }, []);

  return <button onClick={saveFile}>Save</button>;
};
```

```tsx twoslash title="允许文件上传"
import React, { useCallback } from "react";
import { writeStaticFile } from "@remotion/studio";

export const WriteStaticFileComp: React.FC = () => {
  const saveFile = useCallback(
    async (e: React.ChangeEvent<HTMLInputElement>) => {
      const file = e.target.files![0];

      await writeStaticFile({
        filePath: file.name,
        contents: await file.arrayBuffer(),
      });

      console.log("Saved!");
    },
    [],
  );

  return <input type="file" onChange={saveFile} />;
};
```

## 规则

<Step>1</Step> 只能在 Remotion Studio 中使用此 API。
<br />
<Step>2</Step> 文件路径必须相对于<a href="/docs/terminology/public-dir">
  <code>public</code>目录
</a>。
<br />
<Step>3</Step> 不允许在<a href="/docs/terminology/public-dir">
  <code>public</code>目录之外写入文件
</a>。<br />
<Step>4</Step> 要写入子文件夹，请使用正斜杠<code>/</code>，即使在 Windows 上也是如此。
<Step>5</Step> 可以传递<code>string</code>或<code>ArrayBuffer</code>。
