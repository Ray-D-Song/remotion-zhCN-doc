在 Remotion 中，所有动画都必须由 [`useCurrentFrame()`](/docs/use-current-frame) 钩子返回的值驱动。如果您想要在 Remotion 中使用其他动画方式，您需要一个支持与 Remotion 同步时间的集成。

在本页面中，我们维护了一个关于在 Web 上流行的动画方式的集成列表，并提供了流行请求的状态。

## After Effects

查看：[Lottie - 从 After Effects 导入](/docs/after-effects)

## Anime.JS

查看[此存储库](https://github.com/remotion-dev/anime-example) 以获取示例。

## CSS 动画

您可以使用 [`animation-play-state`](https://developer.mozilla.org/en-US/docs/Web/CSS/animation-play-state) 和 [`animation-delay`](https://developer.mozilla.org/en-US/docs/Web/CSS/animation-delay) CSS 属性来同步动画与 Remotion 的时间。

查看[此处的示例](https://stackblitz.com/~/github.com/remotion-dev/css-animation-play-state)。

## Framer Motion

目前，我们没有 Framer Motion 集成，但正在 [GitHub Issues](https://github.com/remotion-dev/remotion/issues/399) 上讨论此事。

## GIF

使用 [`@remotion/gif`](/docs/gif) 包。

## GreenSock

查看：[如何将 GreenSock 与 Remotion 集成](https://archive.ph/dGQ19)

## Lottie

使用 [`@remotion/lottie`](/docs/lottie) 包。

## React Native Skia

使用 [`@remotion/skia`](/docs/skia) 包。

## react-spring

没有直接兼容性，但 Remotion 提供了自己的 [`spring()`](/docs/spring)。

## Reanimated

没有可用的集成，但 Remotion 与 Reanimated 共享一些代码，特别是 [`interpolate()`](/docs/interpolate)、[`spring()`](/docs/spring) 和 [`Easing`](/docs/easing)。这使得从 Reanimated 中重构已有动画更容易。

## Rive

使用 [`@remotion/rive`](/docs/rive) 包。

## TailwindCSS



查看：[TailwindCSS](/docs/tailwind)

## Three.JS

使用 [`@remotion/three`](/docs/three) 包。

## 其他库

您是否有兴趣在 Remotion 中使用其他库？您可以[提交一个 GitHub 问题](https://github.com/remotion-dev/remotion/issues/new)来咨询。虽然我们不能保证能够提供帮助，但您可以注册兴趣并开始讨论。
