表情符号在操作系统中始终按原样呈现。  
它们在不同操作系统上的显示方式各不相同：

- [Windows 表情符号集](https://emojipedia.org/microsoft)
- [macOS 和 iOS 表情符号集](https://emojipedia.org/apple)
- 不同的 Linux 发行版具有不同的表情符号集，或者根本没有。

## Linux 和 Docker 上的表情符号

在云端或一般的 Linux 环境中渲染时，您将无法获得与桌面上相同的表情符号。  
您可以为您的操作系统安装开源表情符号字体：

### Debian/Ubuntu:

```docker
apt-get install fonts-noto-color-emoji
```

### Amazon Linux 2023:

```docker
yum install google-noto-emoji-color-fonts
```

## Remotion Lambda 上的表情符号

[Noto Color Emoji 默认已安装在 Remotion Lambda 上](/docs/lambda/runtime#fonts)。  

您可以[启用 Apple 表情符号](/docs/lambda/deployfunction#runtimepreference)。

通过[自定义 Remotion Lambda](/docs/lambda/custom-layers)可以替换表情符号。

:::note
Apple 表情符号是 Apple Inc. 的知识产权。  
您有责任在您的项目中使用 Apple 表情符号。
:::
