在渲染过程中，Remotion 将项目写入系统的临时目录。这包括渲染帧、未压缩音频和其他工件。

如果默认的临时目录位于可用空间有限的磁盘上，更改临时目录可能会很有用。

如果您想自定义 Remotion 使用的目录，您可以设置 `TMPDIR`（Linux、macOS）或 `TEMP`（Windows）环境变量以指定您喜欢的目录。

```bash
TMPDIR=/var/tmp npx remotion render
```

Remotion 将在您指定的路径中创建一个新的临时目录。
这是因为 Remotion 使用 Node.JS [`os.tmpdir()`][tmpdir] API，该 API 按以下顺序检查环境变量：

* 在非 Windows 平台上为 `TMPDIR`、`TMP`、`TEMP`
* 在 Windows 平台上为 `TEMP` 和 `TMP`

[tmpdir]: https://github.com/nodejs/node/blob/58431c0e6bb1829b6ccafc5cf6340226c15da790/lib/os.js#L181
