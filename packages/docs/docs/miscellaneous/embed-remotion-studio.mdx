我们目前不允许将 Remotion Studio 作为 React 组件嵌入到您自己的应用程序中。这是因为：

<Step>1</Step> 希望嵌入 Studio 通常与自定义 Studio 的外观或行为相关。目前它并不设计为可定制的。<br/>

<Step>2</Step> Studio 与后端连接，它不是一个仅限于前端的应用程序。<br/><br/>

考虑以下替代方案：

## 使用 Remotion Player

Remotion Studio 使用 [Remotion Player](/player)，并围绕其构建了一个 UI。您可以自己使用 Player，并创建与您品牌一致的自定义 UI。

## 部署 Remotion Studio

您可以将 Remotion Studio 部署到云中的长期运行服务器，并使其对您的团队可访问。[点击此处了解如何操作](/docs/studio/deploy-server)。

## 等待 Remotion Bundles 包含 Studio

如果您正在创建一个 [bundle](/docs/bundle)，您将获得一个包含 HTML、CSS 和 JavaScript 文件的静态文件夹。目前，如果您在浏览器中打开此文件夹或将其部署为静态站点，您无法预览视频。我们计划在可预见的未来将 Remotion Studio 包含在 bundle 中，以便您可以在浏览器中预览视频。
