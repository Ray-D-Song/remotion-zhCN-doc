title: Remotion 可以用于直播吗？
sidebar_label: 直播
---

Remotion 不是为广播设计的。它的用例是渲染固定长度的视频，其内容从头到尾是确定的。

直播需要采用不同的方法。内容事先不知道，通常需要能够对事件做出反应，视频长度不固定，广播极其性能敏感，所有帧都需要实时渲染。

Remotion 基于浏览器 API 的声明式方法不适用于这种用例。

对于希望在直播中利用 Web 技术的流媒体用户，我们建议他们构建一个具有透明背景的 Web 应用，并将其作为浏览器源添加到 [OBS](https://obsproject.com/) 中。

您可以使用 Remotion 为 OBS 创建过渡效果，这里有一个 [示例](https://github.com/JonnyBurger/ledevevent-stinger)。
