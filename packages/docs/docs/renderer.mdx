## `renderMedia()` 和 `renderFrames()` 有什么区别？

在 Remotion 3.0 中，我们添加了 [`renderMedia()`](/docs/renderer/render-media) API，它将 `renderFrames()` 和 `stitchFramesToVideo()` 结合为一个简化的步骤，并且执行渲染速度更快。如果可以的话，请优先使用 `renderMedia()`。
