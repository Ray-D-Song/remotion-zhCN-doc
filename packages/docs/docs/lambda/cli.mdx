## 全局选项

### `--region`

选择 AWS 区域，例如：

```
--region=eu-central-1
```

默认区域是 `us-east-1`。您也可以直接设置 `REMOTION_AWS_REGION` 环境变量或通过 `.env` 文件设置。

查看：[区域选择](/docs/lambda/region-selection)

### `--yes`, `-y`

在执行破坏性操作时跳过确认。

### `--quiet`, `-q`

打印最少量的日志。

### `--env-file`

指定 dotenv 文件的位置 - 默认为 `.env`。[了解有关 Remotion 中环境变量的工作方式的信息。](/docs/env-variables)
