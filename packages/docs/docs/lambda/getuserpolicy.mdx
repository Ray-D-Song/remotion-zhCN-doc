返回一个内联 JSON 策略，用于分配给正在用于执行 CLI 命令或调用 Node.JS 函数的 AWS 用户的凭据。

查看[设置教程](/docs/lambda/setup)以从头开始设置 Lambda，或查看[用户权限](/docs/lambda/permissions#user-permissions)以查看带有解释的当前策略文件的副本。

## 示例

```ts twoslash
import { getUserPolicy } from "@remotion/lambda";

console.log(getUserPolicy()); /* `
{
  "Version": "2012-10-17",
  "Statements": [
    // ...
  ]
}
` */
```
