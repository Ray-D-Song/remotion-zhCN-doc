如果在 Remotion Lambda 上渲染视频时遇到以下错误：

```
AccessControlListNotSupported: 存储桶不允许 ACL
```

您正在尝试渲染到一个禁用了 ACL 功能并以不同方式处理其权限的存储桶，例如通过存储桶策略。

## 解决方案

将 `no-acl` 传递给 [`renderMediaOnLambda()`](/docs/lambda/rendermediaonlambda#privacy)、[`renderStillOnLambda()`](/docs/lambda/renderstillonlambda#privacy) 或 [`npx remotion lambda render`](/docs/lambda/cli/render#--privacy) 的 `privacy` 选项。
