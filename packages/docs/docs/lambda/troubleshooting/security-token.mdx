一些 Remotion 支持的区域在 AWS 账户中默认未启用。

如果您收到以下消息：

```
请求中包含的安全令牌无效
```

这意味着该区域在您的 AWS 账户中未启用。

## 选项 1：启用区域

前往 AWS 控制台，然后：

- <Step>1</Step> 点击右上角的您的名称
- <Step>2</Step> 点击 `账户`
- <Step>3</Step> 启用您想要使用的区域。

## 选项 2：使 `getRegions()` 默认仅返回默认启用的区域

使用 [`enabledByDefaultOnly`](/docs/lambda/getregions#enabledbydefaultonly) 选项。
