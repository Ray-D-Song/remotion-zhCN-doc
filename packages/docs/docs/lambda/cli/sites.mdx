`npx remotion lambda sites` 命令允许在您的 S3 存储桶中创建、查看和删除 Remotion 项目。

- [`create`](#create)
- [`ls`](#ls)
- [`rm`](#rm)
- [`rmall`](#rmall)

## create

```
npx remotion lambda sites create <entry-point>?
```

您可以将 [入口点](/docs/terminology/entry-point) 作为第一个参数传递，否则入口点将会被 [确定](/docs/terminology/entry-point#which-entry-point-is-being-used)。

将 Remotion 视频打包并上传到 S3 存储桶。

结果将是一个 URL，例如 `https://remotionlambda-12345.s3.eu-central-1.amazonaws.com/sites/abcdef/index.html`。

:::note
如果您在本地进行更改，您需要重新部署站点。使用 [`--site-name`](#--site-name) 来覆盖现有站点。
:::

您可以使用此“Serve URL”在 Remotion Lambda 上渲染视频：

- 使用 [`npx remotion lambda render`](/docs/lambda/cli/render) 和 [`npx remotion lambda still`](/docs/lambda/cli/still) 命令
- 使用 [`renderMediaOnLambda()`](/docs/lambda/rendermediaonlambda) 和 [`renderStillOnLambda()`](/docs/lambda/renderstillonlambda) 函数。
- 在本地使用 [`renderMedia()`](/docs/renderer/render-media) 和 [`renderStill()`](/docs/renderer/render-still) 函数。
- 在本地使用 [`npx remotion render`](/docs/cli) 和 [`npx remotion still`](/docs/cli) 命令

如果您在 Lambda 上进行渲染，您还可以将站点名称（在本例中为 `abcdef`）作为缩写传递。

### `--region`

选择的[AWS区域](/docs/lambda/region-selection)。项目和函数都应该在此区域内。

### `--site-name`

将项目上传到特定目录并返回确定性URL。如果已经存在具有此名称的站点，则将被覆盖。只能包含以下字符：`0-9`、`a-z`、`A-Z`、`-`、`!`、`_`、`.`、`*`、`'`、`(`、`)`

```
npx remotion lambda sites create src/index.ts --site-name=my-project
```

### `--force-bucket-name`<AvailableFrom v="3.3.42" />

指定要使用的特定存储桶名称。[不建议这样做](/docs/lambda/multiple-buckets)，而是让 Remotion 自动发现正确的存储桶。

### `--privacy`<AvailableFrom v="3.3.97" />

可以是 `public`（默认）或 `no-acl`（如果不使用 ACL）。站点必须具有公共URL才能在Lambda上呈现，因为无头浏览器会打开该URL。

## ls

```
npx remotion lambda sites ls
```

获取站点列表。打印的URL可以传递给 `render` 命令以呈现视频。

<details>
  <summary>示例输出</summary>
  <pre>
    站点名称 存储桶 大小 最后更新时间
    <br />
    pr6fwglz05 remotionlambda-abcdefg 14.7 MB 2021-12-02
    <br />
    https://remotionlambda-abcdefg.s3.eu-central-1.amazonaws.com/sites/pr6fwglz05/index.html
    <br />
    <br />
    testbed remotionlambda-abcdefg 14.7 MB 2021-12-02 <br />
    https://remotionlambda-abcdefg.s3.eu-central-1.amazonaws.com/sites/testbed/index.html
    <br />
  </pre>
</details>

### `--region`

选择的[AWS区域](/docs/lambda/region-selection)。项目和函数都应该在这个区域内。

### `--quiet`, `-q`

仅返回一个以空格分隔的站点列表。

```
npx remotion lambda sites ls -q
```

<details>
  <summary>示例输出</summary>
  <pre>
    pr6fwglz05 testbed
    <br />
  </pre>
</details>

### `--force-path-style`<AvailableFrom v="4.0.202" />

将`forcePathStyle`传递给AWS S3客户端。如果你不知道这是什么，你可能不需要它。
