打印在[设置](/docs/lambda/setup)期间插入AWS控制台的必要权限。

- [`role`](#role)
- [`user`](#user)
- [`validate`](#validate)

:::tip
在 macOS 上，将 `| pbcopy` 添加到命令的末尾以复制输出。
:::

## role

```
npx remotion lambda policies role
```

## validate

遍历所有用户权限并针对AWS策略模拟器对其进行验证。（无法验证角色权限）
