获取此版本的 Remotion Lambda 支持的所有 AWS 区域的数组。

## 示例

```tsx twoslash
// @module: esnext
// @target: es2017
import { getRegions } from "@remotion/lambda";

// ---cut---

const regions = getRegions();
// ["eu-central-1", "us-east-1"]
```

## API

该函数接受一个可选对象，具有以下选项：

### `enabledByDefaultOnly`

_从 v3.3.11 开始提供_

仅返回[在新的 AWS 帐户中默认启用的区域](https://docs.aws.amazon.com/general/latest/gr/rande-manage.html)。

## 返回值

此版本的 Remotion Lambda 支持的区域数组。
