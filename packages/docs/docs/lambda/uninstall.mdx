当然，如果您决定不再使用 Remotion Lambda，我们会感到失望。如果您愿意与我们分享反馈意见，我们将非常乐意倾听。

如果您想要从基础架构中删除**所有与 Remotion Lambda 相关的对象**，您可以按照以下步骤操作。

:::warning
这将删除所有已经渲染的视频，并且会破坏正在使用 Remotion Lambda 渲染视频的程序。
:::

## 删除 Lambda 函数

您可以使用以下命令删除所有函数。`yes` 标志已经包含在内，如果您运行此命令，它将在不需要确认的情况下删除所有函数。

```
npx remotion lambda functions rmall -y
```

## 删除项目

```
npx remotion lambda sites rmall -y
```

## 删除渲染和工件

从您的 AWS 帐户中删除所有以 `remotionlambda-` 开头的 S3 存储桶。

---

确保您已经[将说明应用到所有区域](/docs/lambda/region-selection)。您现在已经清理了您的 AWS 基础架构中的所有与 Remotion Lambda 相关的资源。
