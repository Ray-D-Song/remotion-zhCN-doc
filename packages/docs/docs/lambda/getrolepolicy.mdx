返回一个内联 JSON 策略，用于分配给在您的 AWS 帐户中需要创建的 'remotion-lambda-role' 角色。

这些权限将授予 Lambda 函数本身。

查看[设置教程](/docs/lambda/setup)以从头开始设置 Lambda，或查看[角色权限](/docs/lambda/permissions#role-permissions)以查看带有解释的当前策略文件的副本。

## 示例

```ts twoslash
import { getRolePolicy } from "@remotion/lambda";

console.log(getRolePolicy()); /* `
{
  "Version": "2012-10-17",
  "Statements": [
    // ...
  ]
}
` */
```
