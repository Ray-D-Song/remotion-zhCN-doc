此错误消息**在 Chromium 选项卡崩溃时出现**。继续阅读可能导致此消息的原因。

## 缺少依赖项（Linux 用户）

确保您已在 Linux 上[安装了运行 Chrome 所需的依赖项](/docs/miscellaneous/linux-dependencies)。

## 内存不足或 CPU 过载

如果进程耗尽内存，或者网站因使用 CPU 过重而停止响应，Chrome 选项卡可能会崩溃。

- [减少并发性，以便每个选项卡具有更多资源](/docs/cli/render#--concurrency)
- 在 Remotion Lambda 中，增加内存大小。

## Google Chrome 二进制文件损坏

找到您用于渲染视频的二进制文件。在此计算机上运行测试，看看是否可以正常打开 Google Chrome / Chromium。可能需要重新安装浏览器以解决问题。

:::note
在渲染时添加 `--log=verbose` 以打印正在使用的浏览器可执行文件（从 Remotion v2.6.7 开始）。
:::

这些提示都没有帮助吗？请在 Remotion 存储库中提交问题。
