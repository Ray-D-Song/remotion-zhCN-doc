_从 v2.3 版本开始可用_

Remotion 是一个很好的解决方案，用于渲染视频缩略图或动态静态图像。查看我们的[静态模板](https://github.com/remotion-dev/template-still)，以轻松开始。

如果您已经有一个 Remotion 项目，请继续阅读如何渲染静态图像。

## 定义静态图像

使用 [`<Still />`](/docs/still) 组件来定义静态图像，而不是使用 [`<Composition />`](/docs/composition)。时间轴将消失，您无需定义持续时间或 FPS 值。

## 通过 CLI 渲染

您可以使用 [`npx remotion still`](/docs/cli/still) 命令来渲染静态图像。示例命令：

```bash
npx remotion still --props='{"custom": "data"}' my-comp out.png
```

您可以使用 `--image-format` 标志来确定输出格式。默认格式为 `png`，其他选项包括 `jpeg`、`webp` 和 `pdf`。

默认情况下，将渲染一个组合的帧数，您可以使用 `--frame` 标志来控制它。

## 使用 Node.JS 渲染

您可以使用 [`renderStill()`](/docs/renderer/render-still) Node.JS API 来以编程方式渲染静态帧。

## 使用无服务器渲染

您可以使用 [Remotion Lambda](/lambda) 来渲染静态图像：

- 通过 [CLI](/docs/lambda/cli/still)
- 通过 [renderStillOnLambda](/docs/lambda/renderstillonlambda) API。
