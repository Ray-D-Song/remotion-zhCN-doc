如果您有一个使用 [`<Player>`](/docs/player/player) 组件的 React 应用程序，您可以将其转换为 Remotion 项目，而无需创建新的存储库。这将使您能够：

<Step>1</Step> 使用 <a href="/docs/studio/">Remotion Studio</a>
<br />
<Step>2</Step> 本地渲染视频
<br />
<Step>2</Step> 创建一个允许在云中渲染的 <a href="/docs/terminology/bundle">Remotion Bundle</a>

## 指令

<Step>1</Step> 确保您尚未安装 Studio - 在以下模板中，Studio 已经安装并可以使用命令 <a href="/docs/cli/studio"><code>npx remotion studio</code></a> 启动：
<br />
<br />

- [Next.js (App dir)](/templates/next)
- [Next.js (App dir + TailwindCSS)](/templates/next-tailwind)
- [Next.js (Pages dir)](/templates/next-pages-dir)
- [Remix](/templates/remix)

<Step>2</Step> 安装 Remotion CLI:

<Installation pkg="@remotion/cli" />
<br />

<Step>3</Step> 创建一个名为 <code>remotion</code> 的新文件夹，并在其中添加以下两个文件：
<br />
<br />

```tsx twoslash title="remotion/Root.tsx"
// @filename: Composition.tsx
export const MyComposition: React.FC = () => {
  return null;
};
// @filename: Root.tsx
// ---cut---
import React from 'react';
import {Composition} from 'remotion';
import {MyComposition} from './Composition';

export const RemotionRoot: React.FC = () => {
  return (
    <>
      <Composition
        id="Empty"
        // Import the component and add the properties you had in the `<Player>` before
        component={MyComposition}
        durationInFrames={60}
        fps={30}
        width={1280}
        height={720}
      />
    </>
  );
};
```

```ts twoslash title="remotion/index.ts"
// @filename: Composition.tsx
export const MyComposition: React.FC = () => {
  return null;
};
// @filename: Root.tsx
import React from "react";
import { Composition } from "remotion";
import { MyComposition } from "./Composition";

export const RemotionRoot: React.FC = () => {
  return (
    <>
      <Composition
        id="MyComp"
        component={MyComposition}
        durationInFrames={60}
        fps={30}
        width={1280}
        height={720}
      />
    </>
  );
};
// @filename: index.ts
// ---cut---
import { registerRoot } from "remotion";
import { RemotionRoot } from "./Root";

registerRoot(RemotionRoot);
```

将您先前在 `<Player>` 中注册的组件也添加到 `<Composition>` 中。  
如有必要，将组件移动到 <code>remotion</code> 目录中。

调用 [`registerRoot()`](/docs/register-root) 的文件现在是您的 Remotion [**入口点**](/docs/terminology/entry-point)。

<Step>4</Step> 应用通常在 React 应用程序中启用的常见 Webpack 覆盖：
<br />
<br />

- [TailwindCSS](/docs/tailwind/tailwind)
- [SCSS](/docs/enable-scss/overview)
- [TypeScript 别名](/docs/typescript-aliases#automatically-syncing-webpack-and-typescript-aliases)

## 启动 Remotion Studio:

```
npx remotion studio
```

## 渲染视频 

使用以下命令渲染视频：

```
npx remotion render remotion/index.ts
```

## 设置服务器端渲染 

有关更多信息，请参阅[服务器端渲染](/docs/ssr)。

## 如何从 Remotion Player 下载视频？

首先需要渲染视频 - 这只能在服务器上完成。有关更多信息，请参阅[渲染方式](/docs/render)和[服务器端渲染](/docs/ssr)。
