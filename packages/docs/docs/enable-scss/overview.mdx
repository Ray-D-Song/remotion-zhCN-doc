该包提供了一个 Webpack 覆盖，用于在 Remotion 中启用 [SCSS/SASS](https://sass-lang.com/)。

## 安装

安装 `@remotion/enable-scss` 以及 TailwindCSS 依赖项。

<Installation pkg="@remotion/enable-scss sass@1.77.2 sass-loader@14.2.1 css-loader@5.2.7" />

<br />

:::warning
请注意安装确切的这些版本。  
更新的版本可能无法正常工作。
:::

## 使用

通过使用 [`enableScss()`](/docs/enable-scss/enable-scss) 来[覆盖 Webpack 配置](/docs/webpack)。

```ts twoslash title="remotion.config.ts"
import { Config } from "@remotion/cli/config";
import { enableScss } from "@remotion/enable-scss";

Config.overrideWebpackConfig((currentConfiguration) => {
  return enableScss(currentConfiguration);
});
```
