_v4.0.190版本可用_

这是一个实验性的包，用于解析视频和其他媒体文件，以提取相关的元数据。

:::warning
**不稳定的 API**：此包是实验性的。我们可能随时更改 API，直到我们删除此通知。
:::

设计目标：

- 在浏览器和服务器运行时（Node.js、Bun、Edge 等）中工作
- 流式传输内容，仅在必要时下载尽可能多的数据
- 函数式 TypeScript API
- 解决对 Remotion 用户相关的问题
