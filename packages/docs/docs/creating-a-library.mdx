编写 Remotion 库与编写普通的 React 库相同。通常，它会被发布到 GitHub 和 NPM。

## 起始模板

[Remotion 的库起始模板](https://github.com/remotion-dev/library-starter/) 是创建库或组件以与 Remotion 一起使用的推荐方式。它：

- 正确地将您的库打包为 ESM + CJS
- 内置了 Remotion Studio 实例，用于测试您的库
- 正确地将 Remotion 定义为对等依赖项
- 配置了 TypeScript、ESLint 和 Turborepo

## 发布

您有三种选项来发布您的库：

- **免费发布到 NPM**：起始模板的默认选项。
- **通过 NPM Kiosk 自行销售**：查看更多信息，请参阅[NPM Kiosk](https://remotion.pro/npm-kiosk)。
- **申请在[Remotion 商店](https://www.remotion.pro/store)中列出**：发送电子邮件至 [hi@remotion.dev](mailto:hi@remotion.dev) 进行提案。
