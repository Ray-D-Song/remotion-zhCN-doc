_从 v4.0.187 版本开始可用_

获取可与 [`<AnimatedEmoji>`](/docs/animated-emoji/animated-emoji) 一起使用的可用表情符号列表。

```tsx twoslash title="get-emoji.ts"
import {getAvailableEmojis} from "@remotion/animated-emoji";

const emojiList = getAvailableEmojis();

console.log(emojiList);
```

## 返回值

一个包含以下属性的对象数组：

### `name`

表情符号的名称。您可以将名称传递给 [`emoji`](/docs/animated-emoji/animated-emoji#emoji) 属性。

### `categories`

表情符号所属的类别数组。

### `tags`

表情符号具有的标签数组。

### `durationInSeconds`

表情符号的持续时间（以秒为单位）。

### `codepoint`

表情符号的 Unicode 代码点。
