_来自 [`@remotion/animated-emoji`](/docs/animated-emoji) 包的一部分。_
_从 v4.0.187 开始可用_

显示来自 [Google Fonts Animated Emoji](https://googlefonts.github.io/noto-emoji-animation/) 的动态表情。

:::note
**自托管**: 为了加载资源，您需要将视频从 [`remotion-dev/animated-emoji`](https://github.com/remotion-dev/animated-emoji) 的 `public` 文件夹复制到您的 Remotion 项目的 `public` 文件夹中。
:::

## 示例

```tsx title="Animation.tsx"
import {AnimatedEmoji} from "@remotion/animated-emoji";

export const MyAnimation: React.FC = () => {
  return <AnimatedEmoji emoji="blush" />;
};
```

## 属性

### `emoji`

在此处查看动画版本 [这里](https://googlefonts.github.io/noto-emoji-animation/)。  
默认情况下，不包含任何表情符号资源。请将它们从 [`@remotion/animated-emoji`](/docs/animated-emoji) 的 `public` 文件夹复制到您的 Remotion 项目的 `public` 文件夹中。

### `calculateSrc?`

自定义加载视频的位置。  
这是默认函数:

```tsx twoslash title="calculate-src.ts"
import {staticFile} from "remotion";
import type {CalculateEmojiSrc} from "@remotion/animated-emoji";

export const defaultCalculateEmojiSrc: CalculateEmojiSrc = ({
  emoji,
  scale,
  format,
}) => {
  const extension = format === "hevc" ? "mp4" : "webm";

  return staticFile(`${emoji}-${scale}x.${extension}`);
};
```
