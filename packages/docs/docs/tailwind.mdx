## 在现有项目中安装

_从 v3.3.95 开始，请参阅[旧版本的说明](https://github.com/remotion-dev/remotion/blob/88a5d8d17f50d6ab2b8a408556118d15a3686343/packages/docs/docs/tailwind.md)_

1. 安装 [`@remotion/tailwind`](/docs/tailwind/tailwind) 包和 TailwindCSS 依赖项。

<Tabs
defaultValue="npm"
values={[
{ label: 'npm', value: 'npm', },
{ label: 'yarn', value: 'yarn', },
{ label: 'pnpm', value: 'pnpm', },
]
}>
<TabItem value="npm">

```bash
npm i -D @remotion/tailwind
```

  </TabItem>

  <TabItem value="yarn">

```bash
yarn add -D @remotion/tailwind
```

  </TabItem>
  <TabItem value="pnpm">

```bash
pnpm i -D @remotion/tailwind
```

  </TabItem>
</Tabs>

2. 将 Webpack 覆盖项从 `@remotion/tailwind` 添加到您的配置文件：

```ts twoslash title="remotion.config.ts"
import { Config } from "@remotion/cli/config";
import { enableTailwind } from "@remotion/tailwind";

Config.overrideWebpackConfig((currentConfiguration) => {
  return enableTailwind(currentConfiguration);
});
```

:::note
在 `v3.3.39` 之前，该选项称为 `Config.Bundling.overrideWebpackConfig()`。
:::

3. 如果您使用 [`bundle()` 或 `deploySite()` Node.JS API，请将 Webpack 覆盖项也添加到其中](/docs/webpack#when-using-bundle-and-deploysite)。

4. 创建一个名为 `src/style.css` 的文件，并添加以下内容：

```css title="src/style.css"
@tailwind base;
@tailwind components;
@tailwind utilities;
```

5. 在您的 `src/Root.tsx` 文件中导入样式表。在文件顶部添加：

```js title="src/Root.tsx"
import "./style.css";
```

6. 在项目根目录添加一个 `tailwind.config.js` 文件：

```js title="tailwind.config.js"
/* eslint-env node */
module.exports = {
  content: ["./src/**/*.{ts,tsx}"],
  theme: {
    extend: {},
  },
  plugins: [],
};
```

7. 确保您的 `package.json` 文件中没有设置 `"sideEffects": false`。如果有，请声明 CSS 文件具有副作用：

```diff title="package.json"
{
// Only if `"sideEffects": false` exists in your package.json.
-  "sideEffects": false
+  "sideEffects": ["*.css"]
}
```

8. 开始使用 TailwindCSS！您可以通过向任何元素添加 `className="bg-red-900"` 来验证它是否起作用。

:::note
您的软件包管理器可能会显示对等依赖警告。您可以安全地忽略它，或者在您的 `.npmrc` 中添加 `strict-peer-dependencies=false` 来抑制它。
:::
