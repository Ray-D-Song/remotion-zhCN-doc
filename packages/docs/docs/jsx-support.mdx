自 Remotion 1.3 起，您可以选择退出 Typescript 及其在 Remotion 中的类型检查优势。继续操作需自担风险。

## 选择退出 Typescript

您可以像导入 `.js` 和 `.jsx` 文件一样导入。如果您想完全转换为JS，请将 `index.ts` 和 `Root.tsx` 重命名为 `.jsx` 文件扩展名。删除诸如 `React.FC` 和 `SpringConfig` 等类型。

## 升级

如果您从 Remotion 1.2 或更旧版本升级，请考虑将 `npm test` 命令更改为包括 JavaScript 文件，并删除 `tsc` 命令：

```diff
-  "test": "eslint src --ext ts,tsx && tsc"
+  "test": "eslint src --ext ts,tsx,js,jsx"
```
