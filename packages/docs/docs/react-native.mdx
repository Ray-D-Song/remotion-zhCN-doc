目前正在考虑对 React Native 提供潜在支持。  
2023年5月12日在 App.js Conf 上展示了一个概念验证：

- [存储库](https://github.com/remotion-dev/react-native-demo)
- [演讲](https://youtu.be/t0KDn4_zdrk?t=27073) - 时间戳：7:31:13

## 当前状态

Remotion 正在考虑为 React Native 添加支持，前提是得到 React Native 社区的积极兴趣和关于架构的反馈。

### 目前可用功能:

- 将任何分辨率的视频渲染为 MP4
- `useCurrentFrame()`、`useVideoConfig()`、`spring()`、`interpolate()`、`interpolateColors()` 函数
- 视口外渲染

### 不可用功能:

- 音频支持
- 嵌入式视频
- 多线程渲染
- 除 H264 外的其他编解码器
- 需要 Web API 的 API，例如 `<Sequence>`、`<AbsoluteFill>` 等。
  -> 解决方法：传递 `layout="none"`

## 反馈

在示例存储库上创建问题以提供反馈，或在 [Discord](https://remotion.dev/discord) 的 `#native` 频道上使用。
