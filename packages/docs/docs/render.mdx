### Remotion Studio 部署

可以将 Remotion Studio 部署到云中的长期运行服务器上，然后您的非技术团队成员可以通过 URL 访问。查看 [部署 Remotion Studio](/docs/studio/deploy-server) 指南以了解如何操作。

## CLI

使用 [`render` CLI 命令](/docs/cli/render) 渲染视频：

```bash
npx remotion render HelloWorld
```

修改组合 ID 以选择要渲染的不同视频，或者在末尾添加输出路径以覆盖默认设置。

您可以省略组合名称，将显示选择器：

```bash
npx remotion render
```

## 变体

### 仅音频

您可以选择[仅导出音频](/docs/encoding#audio-only-export)，而不是渲染视频。

### 图像序列

您可以使用[`--sequence`](/docs/cli/render#--sequence)命令将其输出为一系列图像，而不是编码为视频。

### 静态图像

如果您只需要单个图像，可以使用[CLI 或 Node.JS API](/docs/stills)。
