您可以导入此常量以获取 Remotion 的当前版本。

只会报告 [`remotion`](/docs/remotion) 包的版本。  
无法排除与其他 Remotion 包的 [版本冲突](/docs/cli/versions)。

```ts twoslash title="从 remotion 导入 VERSION"
import { VERSION } from "remotion";

console.log(VERSION); // "4.0.57";
```

您还可以从 `remotion/version` 导入它，以避免导入 Remotion 及其依赖项（即 `react` 和 `react-dom`）：

```ts twoslash title="从 remotion/version 导入 VERSION"
import { VERSION } from "remotion/version";

console.log(VERSION); // "4.0.57";
```
