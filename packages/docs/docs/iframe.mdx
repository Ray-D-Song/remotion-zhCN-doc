`<IFrame />` 可以像常规的 `<iframe>` HTML 标签一样使用。

Remotion 会自动将 `<iframe>` 包装在 [`delayRender()`](/docs/delay-render) 调用中，并确保在渲染帧之前加载 iframe。

理想情况下，网站不应该有任何动画，因为 Remotion 仅支持使用 [`useCurrentFrame()`](/docs/use-current-frame) 的动画。请参阅 [Flickering](/docs/flickering) 以获取解释。

## 示例

```tsx twoslash
import { IFrame } from "remotion";

export const MyComp: React.FC = () => {
  return <IFrame src="https://remotion.dev" />;
};
```

## 属性

### `src`

要加载的 URL。

### `delayRenderTimeoutInMilliseconds`<AvailableFrom v="4.0.140" />

自定义此组件调用的 [`delayRender()`](/docs/delay-render) 的[超时](/docs/delay-render#modifying-the-timeout)。

### `delayRenderRetries`<AvailableFrom v="4.0.140" />

自定义此组件调用的 [`delayRender()`](/docs/delay-render) 的[重试次数](/docs/delay-render#retrying)。  
请优先使用 [`maxRetries`](/docs/img#maxretries) 属性。
