当然，如果您决定不再使用 Remotion Cloud Run，我们会感到失望。如果您愿意与我们分享反馈意见，我们将非常乐意接受。

如果您想要从基础架构中**删除所有与 Remotion Cloud Run 相关的对象**，您可以按照以下步骤操作。

:::warning
这将删除所有已渲染的视频，并破坏正在使用 Remotion Cloud Run 渲染视频的程序。
:::

## 删除 Cloud Run 服务

您可以使用以下命令删除所有服务。`yes` 标志已经包含在内，如果运行此命令，将无需确认即可删除所有服务。

```
npx remotion cloudRun services rmall -y
```

## 删除项目

```
npx remotion cloudRun sites rmall -y
```

## 删除渲染和工件

从您的 GCP 项目中删除所有以 `remotioncloudrun-` 开头的 Cloud Storage 存储桶。
