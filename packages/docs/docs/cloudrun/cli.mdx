## 全局选项

### `--region`

选择 GCP 区域：例如：

```
--region=us-central1
```

默认区域是 `us-east1`。您还可以直接设置 `REMOTION_GCP_REGION` 环境变量或通过 `.env` 文件设置。

查看：[区域选择](/docs/cloudrun/region-selection)

### `--yes`, `-y`

在执行破坏性操作时跳过确认。

### `--quiet`, `-q`

打印最少量的日志。
