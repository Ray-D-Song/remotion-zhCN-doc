**优先使用[`<Img />`](/docs/img)或[`<Gif />`](/docs/gif)** 而不是原生的`<img>`标签、Next.js中的`<Image>`和CSS的`background-image`。  
**优先使用[`<OffthreadVideo />`](/docs/offthreadvideo)或[`<Video />`](/docs/video)** 而不是原生的`<video>`标签。  
**优先使用[`<Audio />`](/docs/audio)** 而不是原生的`<audio>`标签。  
**优先使用[`<IFrame />`](/docs/iframe)** 而不是原生的`<iframe>`标签。

<br/>

通过使用Remotion的组件，您可以确保：

<Step>1</Step> 在渲染帧之前完全加载资产<br/>
<Step>2</Step> 图像和视频与Remotion的时间轴同步。
