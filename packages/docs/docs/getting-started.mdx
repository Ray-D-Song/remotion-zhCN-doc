## 先决条件

要使用 Remotion，您至少需要 [Node](https://nodejs.org/en/download/) <MinNodeVersion /> 或 [Bun](https://bun.sh) <MinBunVersion />。

## 创建一个新项目

:::note
🐧 [Linux 发行版至少需要 2.35 版本的 Libc。](https://github.com/remotion-dev/remotion/issues/2439)  
它们还需要[安装一些额外的软件包](/docs/miscellaneous/linux-dependencies)。
:::
