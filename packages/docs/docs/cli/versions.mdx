_从 Remotion v2.6.2 开始可用_

打印出已安装的 Remotion 包的版本。

```
npx remotion versions
```

在最佳情况下，您的版本是对齐的，命令的输出如下所示：

```
On version: 3.0.19
- @remotion/bundler
- @remotion/cli
- @remotion/eslint-config
- @remotion/renderer
- @remotion/media-utils
- @remotion/babel-loader
- @remotion/lambda
- @remotion/three
- @remotion/gif
- remotion

✅ Great! All packages have the same version.
```
