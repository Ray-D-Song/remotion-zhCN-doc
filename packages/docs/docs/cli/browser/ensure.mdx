# npx remotion 浏览器 ensure<AvailableFrom v="4.0.137" />

确保 Remotion 有一个可用于渲染的浏览器。

```
npx remotion browser ensure
```

## 参数

### `--browser-executable`

[自定义 Chrome 可执行文件的路径](/docs/config#setbrowserexecutable)。如果未指定且 Remotion 找不到可用的浏览器，此命令将下载一个。
