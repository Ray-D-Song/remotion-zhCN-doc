_在 v4.0.0 中已移除，从 v3.3 开始可用_

_查看["不再需要安装 FFmpeg"](https://translate.google.com/docs/4-0-migration#no-more-ffmpeg-install-ffmpegexecutable-option-removed)._  
此页面仅供存档目的。

通过从互联网下载它们来确保已安装 `ffmpeg` 或 `ffprobe`，如果找不到它们。

```bash
npx remotion install ffmpeg
```

```bash
npx remotion install ffprobe
```

您可能不需要调用此函数。如果尝试渲染并且找不到二进制文件，则 Remotion 将自动下载 `ffmpeg` 和 `ffprobe`。

如果需要在第一次渲染开始之前准备好 `ffmpeg` 和 `ffprobe`，则这些命令很有用。
