_自 v4.0 起可用_

为了能够使用 `ffprobe` 而无需直接安装它，Remotion 通过 `npx remotion ffprobe` 提供了它。

请注意，为了保持二进制文件的大小，FFprobe 二进制文件仅支持 Remotion 本身支持的编解码器：H.264、H.265、VP8、VP9 和 ProRes。使用的是来自 FFprobe 6.0 发行线的二进制文件。

# 示例

```bash
npx remotion ffprobe your_video.mp4
```

要了解更多关于 FFprobe 的信息，请访问他们的[文档](https://ffmpeg.org/ffprobe.html)。
