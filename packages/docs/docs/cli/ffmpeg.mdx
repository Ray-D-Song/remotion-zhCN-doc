_自 v4.0 起可用_

为了能够使用 `ffmpeg` 而无需直接安装它，Remotion 通过 `npx remotion ffmpeg` 提供了它。

请注意，为了保持二进制文件的大小，FFmpeg 二进制文件仅支持 Remotion 本身支持的编解码器：H.264、H.265、VP8、VP9 和 ProRes。使用的是 FFmpeg 6.0 发行线的二进制文件。

# 示例

将视频文件转换为音频文件

```bash
npx remotion ffmpeg -i input.mp4 output.mp3
```

要了解更多关于 FFmpeg 的信息，请访问他们的 [文档](https://ffmpeg.org/documentation.html)。
