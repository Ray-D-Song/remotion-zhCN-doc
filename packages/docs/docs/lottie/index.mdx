该包提供了一个组件，用于在 Remotion 中显示 [Lottie](http://airbnb.io/lottie/) 动画。

## 安装

安装 `@remotion/lottie` 以及 `lottie-web`。

现在您可以在您的 Remotion 项目中开始使用 [`<Lottie>`](/docs/lottie/lottie) 组件。

## 支持的功能

- 使用 `lottie-web` 播放 Lottie 动画
- 更改动画的速度
- 正向和反向播放动画
- 播放远程文件
- 确定 Lottie 动画的尺寸和持续时间

## 不支持的功能

- 在其他渲染器上呈现为 `svg`
- `setSubFrame()`，`setLocationHref()`
- 有限的表达式支持：Remotion 使用 `lottie-web` 的 `.goToAndStop()` 方法来浏览 Lottie 文件。根据表达式的不同，帧可能不会以确定性方式呈现，导致在 Remotion 输出中出现 [闪烁](/docs/flickering)。Remotion 无法在不改变 `lottie-web` 上游的情况下解决这个问题。您需要根据具体情况评估您正在使用的表达式是否受 Remotion 支持。

:::note
如果您想要请求当前不支持的功能，请[提交一个问题](https://remotion.dev/issue)。
:::
