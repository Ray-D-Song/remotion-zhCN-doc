_属于[`@remotion/google-fonts`](/docs/google-fonts)包_

每种字体都导出一个`getInfo()`函数，可用于在运行时获取有关字体的各种信息，例如可用的权重、样式和子集。

## 用法

要获取有关字体的信息，您可以导入`info`属性：

```tsx twoslash title="获取字体信息"
import { getInfo } from "@remotion/google-fonts/Montserrat";
console.log(getInfo());
```
