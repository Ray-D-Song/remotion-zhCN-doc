_属于[`@remotion/google-fonts`](/docs/google-fonts)包的一部分_

`@remotion/google-fonts`中所有可用字体的数组。  
从v3.3.64开始，如果您正在使用ES模块加载函数，则可以通过调用`.load()`来单独加载字体。

## 用法

```ts twoslash
import { getAvailableFonts } from "@remotion/google-fonts";

console.log(getAvailableFonts());
```

```ts title="结构（缩短）"
[
  {
    fontFamily: "ABeeZee",
    importName: "ABeeZee",
    load: () => import("./ABeeZee"), // Available from v3.3.64
  },
  {
    fontFamily: "Abel",
    importName: "Abel",
    load: () => import("./Abel"),
  },
  {
    fontFamily: "Abhaya Libre",
    importName: "AbhayaLibre",
    load: () => import("./AbhayaLibre"),
  },
];
```

- `fontFamily`是在CSS中引用名称的方式。
- `importName`是字体可以被导入的标识符：`@remotion/google-fonts/[import-name]`。

## 关于CommonJS的注意事项

如果您使用`require()`导入此模块，则无法动态加载字体。这也适用于转译为在幕后使用`require()`的代码。较新版本的Next.js、Vite和Astro已自动配置为允许延迟加载。
