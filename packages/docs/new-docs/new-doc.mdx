title: 一个新的文档页面
---

在这里写下你的文章并将其发送为一个拉取请求！  
Remotion 团队将为您对其进行分类。

要编写一个经过类型检查的代码片段，请使用 <code>```tsx twoslash</code> 开始一个代码块：

```tsx twoslash title="MyComponent.tsx"
import {useCurrentFrame} from 'remotion';

const frame = useCurrentFrame();
```

您可以使用 `<Step>` 创建编号列表和章节

<Step>1</Step> 一个 <br/>
<Step>2</Step> 二 <br/>

<br/>

对于段落中的编号项目，请使用 `<InlineStep>`: <InlineStep>1</InlineStep> 一个, <InlineStep>2</InlineStep> 二.

查看 [语言指南](/docs/contributing/docs#language-guidelines) 以便您的文章能够很好地融入到网站上其他可用的文章中。
